Lightning talk

Lightning talk，中文譯為「閃電秀」，顧名思義即為「閃電一般短的演講」。包含準備時間在內，每位講者僅有 3 分鐘的時間上台。用飛快的速度進行簡潔扼要的分享，無論是對台上的講者或是台下的會眾，都是十分刺激又轟動全場的體驗。Lightning talk 將會在年會當天早上於白板區開放報名，先搶先贏，並在所有議程結束後，閉幕前，在國際會議廳進行演講。

沙龍

SITCON 每年必定會有的沙龍，今年也不從缺！14:10～15:10 的交誼廳裡，無論是校園生活、工作經驗、技術分享，或是更多你意想不到的內容，都可能是沙龍輕鬆聊天討論的主題。你可能會碰到不同學校、不同科系的同學和 SITCON 工作人員，甚至是捕獲到野生的講者！來自四面八方的資訊同好可以在沙龍裡激盪出什麼精彩的內容呢？期待你一同來沙龍認識新朋友，分享自己的故事！

Unconference

今年特別誕生了完全由會眾決定議程的 Unconference！

如果你有滿腹主題想要跟大家分享，可是先前投稿不幸落選，閃電秀又無法讓你暢所欲言，那麼參與 Unconference 投稿徵選就是你最佳選擇！只要記得在 2/12 前投稿，並有足夠的會眾投票支持，就會獲得二十分鐘的時間能上台分享。

而又同時身為會眾的你，若有想聽的待選 Unconference 議程，也別忘了在 2/13 ～ 2/21 投票支持。

入選的 Unconference 將在 2/22 公佈，敬請期待！

  1. 徵選和票選 Unconference 議程都需以 SITCON 票卷的 KKTIX 檢查碼為憑。
  2. 議程組會初步審核 Unconference 內容，除非有重大違規，否則議程組不會干涉 Unconference 投稿內容。
  3. Unconference 議程的品質議程組不予負責。

天使計畫

天使計畫是一個提供會眾們一對一交流機會的活動。在此誠摯地邀請曾參加過年會的夥伴擔任「天使」，來幫助 SITCON 新手「初心者」更迅速的了解並融入 SITCON 社群！

在報名表單上勾選參與並選擇與對方的聯絡方式，之後年會將會隨機配對天使與初心者，並用電子信箱告知配對結果。屆時雙方便可熟悉彼此，並在年會當天能有更深一層的當面交流！

  1. 信箱必須提供主辦方與配對方，以利通知配對結果。
  2. 年會當天將設有建議的會面地點以便相認。
  3. 主辦方不保證配對成功，亦不處理配對後的相處狀況。
  4. 若當天有任何問題請向工作人員反映。

程式拼圖

破碎的程式，需要你與大家合作一起完成！

每人將會在入場時發給一個零碎的程式碼片段，需要尋覓其他夥伴一同合作拼湊，將合適的片段組成一支完整的程式，再交給 Compiler 進行編譯。只要輸出正確的題目解答，就完成了程式拼圖，並可獲得 SITCON 精美小禮物哦！

  1. 題目有五題，請擇一解題即可。
  2. 每段程式碼可重複使用。會場內設有活動區，供會眾找尋夥伴，但活動不僅限於活動區內進行，會眾也可以用任何方式尋找夥伴。
  3. 有任何活動相關疑問皆可向 Debugger 攤位詢問。但除了規則及活動本身相關問題外，包含尋找夥伴等解題問題，請盡量向身邊夥伴交流解決。
  4. 贊助商也有一些程式碼片段，歡迎前往贊助商攤位詢問。

攤位

SITCON 今年新增了攤位活動，目的是在促進會眾跟學生社團或相關資訊社群交流。例如向女孩推廣程式的 Django Girls 等社群或是各校資訊社團。但除此之外，更有開鎖達人的不藏私分享、香味四溢的視界咖啡館等非資訊相關的攤位一同參加。如此多元的活動讓你在聽完議程意猶未盡之餘，更能充分交流！休息時間，別忘記到場外的攤位逛逛！
